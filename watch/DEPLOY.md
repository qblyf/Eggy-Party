# 部署指南 - 小天才手表版

## 🚀 快速部署（3分钟）

### 使用 Vercel（最简单）

1. **注册 Vercel**
   - 访问 https://vercel.com
   - 使用 GitHub 账号登录

2. **导入项目**
   - 点击 "New Project"
   - 导入你的 GitHub 仓库
   - 或直接拖拽 `watch` 文件夹

3. **配置**
   - Root Directory: 选择 `watch`
   - 其他保持默认
   - 点击 Deploy

4. **获得网址**
   - 部署完成后自动生成网址
   - 例如：`https://your-project.vercel.app`

### 使用 GitHub Pages

1. **创建仓库**
   ```bash
   # 在项目根目录
   git init
   git add .
   git commit -m "Initial commit"
   ```

2. **推送到 GitHub**
   ```bash
   # 创建 GitHub 仓库后
   git remote add origin https://github.com/你的用户名/仓库名.git
   git branch -M main
   git push -u origin main
   ```

3. **开启 Pages**
   - 进入仓库 Settings
   - 左侧找到 Pages
   - Source 选择 `main` 分支
   - 文件夹选择 `/watch` 或 `/ (root)`
   - 点击 Save

4. **访问网址**
   - 等待 1-2 分钟
   - 访问：`https://你的用户名.github.io/仓库名/watch/`

### 使用 Netlify

1. **注册 Netlify**
   - 访问 https://netlify.com
   - 使用 GitHub 登录

2. **拖拽部署**
   - 点击 "Add new site" > "Deploy manually"
   - 直接拖拽 `watch` 文件夹
   - 自动部署

3. **获得网址**
   - 自动生成随机网址
   - 可以自定义域名

## 📱 生成二维码

### 方法一：草料二维码（推荐）

1. 访问 https://cli.im
2. 输入你的网址
3. 点击生成
4. 下载二维码图片
5. 打印或保存到手机

### 方法二：在线工具

- https://www.qrcode-monkey.com/
- https://www.the-qrcode-generator.com/
- 微信小程序搜索"二维码生成器"

### 方法三：使用命令行

```bash
# 安装 qrcode
npm install -g qrcode

# 生成二维码
qrcode "https://你的网址.com/watch/" -o qrcode.png
```

## 🔗 短链接生成

为了方便手表输入，建议生成短链接：

### 推荐服务

1. **短链接服务**
   - https://suo.im （国内）
   - https://bitly.com （国际）
   - https://tinyurl.com

2. **生成步骤**
   - 输入完整网址
   - 生成短链接
   - 例如：`suo.im/abc123`

3. **优势**
   - 易于记忆
   - 方便输入
   - 可以修改目标网址

## 🎯 小天才手表打开方式

### 方式一：扫码（最方便）⭐

```
1. 生成二维码
2. 手表打开"扫一扫"
3. 扫描二维码
4. 自动打开游戏
```

### 方式二：发送链接

```
1. 家长APP发送链接
2. 手表接收消息
3. 点击链接打开
```

### 方式三：语音输入

```
1. 对手表说"打开浏览器"
2. 语音输入短网址
3. 或手动输入
```

### 方式四：添加到桌面

```
1. 首次用上述方式打开
2. 浏览器菜单 > 添加到主屏幕
3. 下次直接从桌面打开
```

## 🔧 故障排除

### 网址打不开

**问题**：手表无法访问网址

**解决**：
1. 检查手表是否联网
2. 确认网址拼写正确
3. 尝试用手机先打开测试
4. 检查部署是否成功

### 页面显示异常

**问题**：页面布局错乱

**解决**：
1. 清除手表浏览器缓存
2. 重新打开网页
3. 检查是否是最新版本

### 图片不显示

**问题**：卡片图片加载失败

**解决**：
1. 检查图片路径是否正确
2. 确认图片文件已上传
3. 会自动显示 emoji 替代
4. 可以只使用 emoji 版本

### 无法添加到桌面

**问题**：找不到"添加到主屏幕"选项

**解决**：
1. 确认手表浏览器支持 PWA
2. 尝试不同的浏览器
3. 可以添加书签代替
