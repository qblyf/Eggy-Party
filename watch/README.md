# 蛋仔盲盒 - 小天才手表版

## 📱 如何在小天才手表上打开

### 方法一：扫码打开（推荐）✅

1. **部署网站**
   - 将 `watch` 文件夹上传到服务器
   - 或使用 GitHub Pages / Vercel 等免费托管
   - 获得网址，例如：`https://你的域名.com/watch/`

2. **生成二维码**
   - 使用草料二维码（cli.im）或其他工具
   - 输入网址生成二维码
   - 打印或保存二维码图片

3. **手表扫码**
   - 打开小天才手表的"扫一扫"功能
   - 扫描二维码
   - 自动在手表浏览器中打开游戏

### 方法二：手机发送链接 📲

1. **在手机APP中操作**
   - 打开小天才APP（家长端）
   - 找到"发消息"或"聊天"功能
   - 发送网址链接给手表

2. **手表接收**
   - 手表收到消息后
   - 点击链接即可打开游戏

### 方法三：语音输入网址 🎤

1. **使用语音助手**
   - 对手表说："打开浏览器"
   - 使用语音输入网址
   - 或手动输入短网址

2. **建议使用短网址**
   - 使用短链接服务（如 suo.im）
   - 将长网址转换为短网址
   - 例如：`suo.im/abc123`

### 方法四：添加到桌面（PWA）⭐

1. **首次打开后**
   - 在手表浏览器中打开游戏
   - 点击浏览器菜单（通常是三个点）
   - 选择"添加到主屏幕"或"添加到桌面"

2. **下次使用**
   - 直接从手表桌面点击图标
   - 像APP一样打开游戏
   - 支持离线使用

## 📦 部署方法

### 使用 GitHub Pages（免费）

1. **创建仓库**
   ```bash
   git init
   git add watch/
   git commit -m "Add watch version"
   git branch -M main
   git remote add origin https://github.com/你的用户名/仓库名.git
   git push -u origin main
   ```

2. **开启 Pages**
   - 进入仓库 Settings
   - 找到 Pages 选项
   - Source 选择 main 分支
   - 保存后获得网址

3. **访问地址**
   - `https://你的用户名.github.io/仓库名/watch/`

### 使用 Vercel（免费）

1. **安装 Vercel CLI**
   ```bash
   npm i -g vercel
   ```

2. **部署**
   ```bash
   cd watch
   vercel
   ```

3. **获得网址**
   - 自动生成网址
   - 支持自定义域名

## 📝 注意事项

1. **图片路径**
   - 图片使用相对路径 `../assets/images/`
   - 确保图片文件存在
   - 图片加载失败会显示 emoji

2. **浏览器兼容**
   - 支持现代浏览器
   - 小天才手表浏览器基于 WebKit
   - 建议使用 Chrome 测试

3. **数据存储**
   - 使用 localStorage 保存进度
   - 清除浏览器数据会丢失进度
   - 建议定期截图保存收藏

4. **网络要求**
   - 首次访问需要网络
   - 安装 PWA 后可离线使用
   - 图片需要网络加载

## 🆘 常见问题

**Q: 手表打不开网页？**
- 检查手表是否联网
- 确认网址是否正确
- 尝试重启手表浏览器

**Q: 游戏卡顿？**
- 关闭其他应用
- 重启手表
- 清理浏览器缓存

**Q: 进度丢失？**
- 不要清除浏览器数据
- 不要使用无痕模式
- 建议截图保存

**Q: 图片不显示？**
- 检查网络连接
- 确认图片路径正确
- 会自动显示 emoji 替代
