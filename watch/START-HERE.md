# 🎉 从这里开始 - 蛋仔盲盒小天才手表版

欢迎！这是专门为小天才儿童手表优化的蛋仔盲盒游戏 PWA 版本。

```
watch/
├── index.html          # 游戏主页面
├── style.css           # 样式文件（已优化小屏幕）
├── script.js           # 游戏逻辑
├── manifest.json       # PWA 配置
├── sw.js              # Service Worker（离线支持）
```

## 🚀 快速开始（5分钟）

### 第1步：本地测试（1分钟）

```bash
# 方法1：直接打开
双击 test.html 文件
# 显示 320×360 方形手表屏幕

# 方法2：启动服务器（推荐）
cd watch
python3 -m http.server 8080
# 然后打开 http://localhost:8080
```

### 第2步：部署到网上（3分钟）

**推荐使用 Vercel（最简单）：**

1. 访问 https://vercel.com
2. 用 GitHub 登录
3. 点击 "New Project"
4. 拖拽 `watch` 文件夹上传
5. 点击 "Deploy"
6. 获得网址（例如：`https://xxx.vercel.app`）

详细步骤看 → `DEPLOY.md`

### 第3步：在手表上打开（1分钟）

**最简单的方法 - 扫码：**

1. 用草料二维码生成器（https://cli.im）
2. 输入你的网址
3. 生成二维码
4. 手表"扫一扫"打开

详细方法看 → `小天才使用指南.md`

## 📱 小天才手表如何打开？

### 🌟 方法1：扫码（推荐）
- 生成二维码

- 家长APP发送链接
- 手表点击打开

### 🎤 方法3：语音输入
- 对手表说"打开浏览器"
- 输入网址

### ⭐ 方法4：添加到桌面
- 首次打开后
- 选择"添加到主屏幕"
- 像APP一样使用

## 🎮 游戏说明

### 如何玩

1. 点击盲盒或按钮抽取蛋仔
2. 收集不同的蛋仔角色
3. 查看收藏和完成度
4. 目标：收集所有13个蛋仔

### 稀有度

- **普通** (60%) - 常见蛋仔
- **稀有** (30%) - 比较少见
- **超稀有** (9%) - 很难得到
- **传说** (1%) - 超级稀有的黄金蛋仔！

## 🔧 技术栈

- HTML5 + CSS3 + JavaScript
- PWA (Progressive Web App)
- Service Worker（离线支持）
- LocalStorage（数据保存）
- 响应式设计

## ❓ 常见问题

### Q: 需要编程知识吗？
A: 不需要！按照文档操作即可。

### Q: 完全免费吗？
A: 是的！使用 Vercel 或 GitHub Pages 完全免费。

### Q: 需要多长时间？
A: 从部署到在手表上玩，大约5-10分钟。

### Q: 手表一定能打开吗？
A: 只要手表有浏览器功能就可以。小天才手表都支持。

### Q: 数据会丢失吗？
A: 不会，自动保存在手表本地。但不要清除浏览器数据。

### Q: 可以修改游戏吗？
A: 可以！代码完全开源，可以自由修改。
