# 📝 更新日志

## v2.0 - 滑动版本 (2025-01-18)

### 🎉 重大更新：双页面滑动设计

#### 新增功能

1. **左右滑动切换页面** ⭐
   - 将抽取和收藏分成两个独立页面
   - 支持左右滑动切换
   - 更符合手表操作习惯

2. **页面指示器**
   - 底部显示当前页面位置
   - 可点击快速切换页面
   - 半透明毛玻璃效果

3. **滑动提示**
   - 每个页面显示滑动提示
   - 动画效果引导用户
   - 自动淡入淡出

4. **独立收藏页面**
   - 专门的收藏展示页面
   - 独立的统计卡片
   - 更清晰的布局

5. **鼠标拖拽支持**
   - 电脑上可以用鼠标测试
   - 模拟触摸滑动
   - 方便开发调试

#### 改进优化

1. **操作体验**
   - 左右滑动比上下滚动更自然
   - 智能阈值判断（30%屏幕宽度）
   - 平滑的动画过渡
   - 自动回弹效果

2. **视觉设计**
   - 两个页面不同的渐变背景
   - 抽取页面：金黄→粉红（温暖）
   - 收藏页面：紫蓝→深紫（优雅）
   - 更好的视觉区分

3. **性能优化**
   - 使用 transform 而非 position
   - 启用硬件加速
   - 流畅的 60fps 动画
   - 低内存占用

4. **布局优化**
   - 每个页面独立滚动
   - 避免长页面滚动
   - 内容一目了然
   - 减少误操作

#### 技术改进

1. **触摸事件处理**
   - touchstart/touchmove/touchend
   - 实时跟随手指
   - 流畅的拖拽体验

2. **鼠标事件支持**
   - mousedown/mousemove/mouseup
   - 电脑端测试支持
   - 与触摸体验一致

3. **状态管理**
   - 记录当前页面
   - 同步更新指示器
   - 正确的页面切换

4. **CSS 优化**
   - 使用 will-change 提示
   - cubic-bezier 缓动函数
   - 减少重绘和回流

## 🎯 升级建议

### 从 v1.0 升级到 v2.0

1. **备份数据**
   - 游戏数据保存在 localStorage
   - 升级不会丢失数据
   - 建议截图保存收藏

2. **替换文件**
   - 下载新版本文件
   - 替换旧版本
   - 重新部署

3. **测试功能**
   - 测试滑动功能
   - 确认数据正常
   - 检查所有功能

4. **更新二维码**
