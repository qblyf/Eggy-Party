# 🥚 蛋仔盲盒 - 小天才手表版

## 📁 项目结构

```
蛋仔派对抽盲盒/
├── 原版游戏文件
│   ├── index.html
│   ├── style.css
│   ├── script.js
│   └── assets/
│
└── watch/                    # 👈 小天才手表优化版
    ├── 🎮 游戏文件
    │   ├── index.html        # 主页面
    │   ├── style.css         # 样式（已优化小屏幕）
    │   ├── script.js         # 游戏逻辑
    │   ├── manifest.json     # PWA配置
    │   ├── sw.js            # Service Worker
    │   └── test.html        # 测试页面
    │
    ├── 📖 使用文档
    │   ├── START-HERE.md           # 👈 从这里开始！
    │   ├── 小天才使用指南.md        # 如何在手表上打开
    │   ├── DEPLOY.md              # 部署教程
    │   ├── 本地测试.md             # 测试指南
    │   ├── 完成清单.md             # 检查清单
    │   └── README.md              # 完整说明
    │
    └── 🎨 图标相关
        ├── icon-guide.md          # 图标制作教程
        ├── icon.svg              # SVG图标（备用）
        └── ICONS-NEEDED.txt      # 需要的图标说明
```

## 🚀 快速开始

### 1️⃣ 打开文档
```
📂 watch/START-HERE.md
```
这是最重要的文档，包含所有你需要的信息！

### 2️⃣ 本地测试
```bash
# 进入 watch 目录
cd watch

# 双击打开 test.html
# 或启动服务器
python3 -m http.server 8080
```

### 3️⃣ 部署上线
按照 `watch/DEPLOY.md` 的步骤部署到 Vercel 或 GitHub Pages

### 4️⃣ 在手表上打开
按照 `watch/小天才使用指南.md` 的方法在小天才手表上打开

## 📱 小天才手表如何打开？

### 最简单的方法：扫码 ⭐

1. 部署网站获得网址
2. 用草料二维码（cli.im）生成二维码
3. 小天才手表"扫一扫"
4. 自动打开游戏
5. 添加到桌面，像APP一样使用

详细方法请看：`watch/小天才使用指南.md`

## ❓ 常见问题

### Q: 需要编程知识吗？
**A:** 不需要！按照文档操作即可。

### Q: 完全免费吗？
**A:** 是的！使用 Vercel 或 GitHub Pages 完全免费。

### Q: 需要多长时间？
**A:** 从部署到在手表上玩，大约5-10分钟。

### Q: 所有小天才手表都能用吗？
**A:** 只要有浏览器功能就可以。大部分型号都支持。

### Q: 数据会丢失吗？
**A:** 不会，自动保存在本地。但不要清除浏览器数据。

### Q: 可以修改游戏吗？
**A:** 可以！代码完全开源，可以自由修改。

更多问题请查看各个文档的"常见问题"部分。
